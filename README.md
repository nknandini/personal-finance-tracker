# Personal Finance Tracker

A Python-based finance tracking application designed to record, analyze, and visualize personal spending patterns using data analytics concepts.

## 📌 Project Overview
This project helps users manage and monitor expenses efficiently by categorizing spending, generating summaries, visualizing trends, and predicting future expenses using basic machine learning techniques.

The project was developed to strengthen my understanding of Python programming, data analysis, visualization, and predictive analytics.

---

## ✨ Features
- Add and manage expenses
- Categorize spending records
- View category-wise spending summaries
- Monthly spending analysis
- Budget monitoring
- Spending trend visualization using graphs
- Predict next expense using Linear Regression

## 🚀 How to Run the Project

1. Clone the repository:
```bash
git clone https://github.com/nknandini/personal-finance-tracker.git
```
2. Install required libraries:
```bash
pip install pandas numpy matplotlib scikit-learn
```
3. Run the Python file:
```bash
python finance_tracker.py
```
